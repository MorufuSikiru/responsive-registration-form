# 📝 Responsive Registration Form

A responsive registration form built with HTML, CSS, and JavaScript, designed to collect user information with proper validation and a clean user interface.

## 🔧 Features

- Responsive design (mobile & desktop)
- Input validation (email, password, etc.)
- Error message display
- Clean and modern UI

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript
